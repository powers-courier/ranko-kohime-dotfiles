** Performance
*** Limit the application FPS.
Comma-separated list of one or more FPS values (e.g. 0,30,60). 0 means unlimited (unless VSynced).
#+begin_src conf
# fps_limit=
#+end_src

*** VSync
[0-3] 0 = adaptive; 1 = off; 2 = mailbox; 3 = on
#+begin_src conf
# vsync=
#+end_src

*** OpenGL VSync
[0-N] 0 = off; >=1 = wait for N v-blanks, N > 1 acts as a FPS limiter (FPS = display refresh rate / N)
#+begin_src conf
# gl_vsync=
#+end_src
