*** Proxy Hosts
**** Proxy Host Generator
***** Explanation
****** proxyCount:
Set to 3—generates jelly-proxy-01, jelly-proxy-02, jelly-proxy-03.
Change to 5 for jelly-proxy-05, etc.
****** builtins.genList:
builtins.genList (i: i) 3 → [0 1 2]—a list of indices.
i + 1 shifts it to 1, 2, 3 for numbering.
****** map:
Loops over [0 1 2], creating a list of { name = ...; value = ...; } pairs.
num: Pads with 0 for 01-09 (e.g., if i < 9 then "0${toString (i + 1)}"), then plain numbers (10, 11, ...).
****** builtins.listToAttrs:
Turns the list into an attrset:
****** networking.hostName = name:
Inside the inline module, name (e.g., "jelly-proxy-01") sets the hostName dynamically—no repetition.
****** // Operator:
Merges proxyConfigs with other configs (e.g., jellyfin-db)—keeps non-proxy hosts separate.
***** Code
#+begin_src nix :noweb-ref proxy-host-generator
proxyCount = 1;
Jelly-Proxy-Configs = builtins.listToAttrs (map (i: let
  num = if i < 9 then "0${toString (i + 1)}" else toString (i + 1);
  name = "jelly-proxy-${num}";
in {
  inherit name;
  value = nixpkgs.lib.nixosSystem {
    system = "x86_64-linux";
    specialArgs = { inherit vars; };
    modules = [
      ({ config, pkgs, vars, ... }: {
        networking.hostName = name;
        system.stateVersion = "25.05";
      })
      ./hosts/${name}-hardware.nix
      ./modules/default-modules.nix
      ./hosts/remote-proxy-nodes.nix
    ];
  };
}) (builtins.genList (i: i) proxyCount));
#+end_src
