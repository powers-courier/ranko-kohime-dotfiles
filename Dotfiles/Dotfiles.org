**** Commentary
List of special keys: --input-keylist
Keybindings testing mode: mpv --input-test --force-window --idle

Use 'ignore' to unbind a key fully (e.g. 'ctrl+a ignore').

Strings need to be quoted and escaped:
  KEY show-text "This is a single backslash: \\ and a quote: \" !"

You can use modifier-key combinations like Shift+Left or Ctrl+Alt+x with
the modifiers Shift, Ctrl, Alt and Meta (may not work on the terminal).

The default keybindings are hardcoded into the mpv binary.
You can disable them completely with: --no-input-default-bindings

Developer note:
On compilation, this file is baked into the mpv binary, and all lines are
uncommented (unless '#' is followed by a space) - thus this file defines the
default key bindings.

**** Default Bindings
If this is enabled, treat all the following bindings as default.
#+begin_src conf
default-bindings start
#+end_src

**** Mouse Buttons
#+begin_src conf
MBTN_LEFT     ignore              # don't do anything
MBTN_LEFT_DBL cycle fullscreen    # toggle fullscreen
MBTN_RIGHT    cycle pause         # toggle pause/playback mode
MBTN_BACK     playlist-prev       # skip to the previous file
MBTN_FORWARD  playlist-next       # skip to the next file
#+end_src

**** Mouse Wheel / Touchpad axis input
Mouse wheels, touchpad or other input devices that have axes
if the input devices supports precise scrolling it will also scale the
numeric value accordingly
#+begin_src conf
WHEEL_UP      seek 10          # seek 10 seconds forward
WHEEL_DOWN    seek -10         # seek 10 seconds backward
WHEEL_LEFT    add volume -2
WHEEL_RIGHT   add volume 2
#+end_src

**** Seeking
***** Standard Seeking
Seek units are in seconds, but note that these are limited by keyframes
#+begin_src conf
RIGHT seek  5
LEFT  seek -5
UP    seek  60
DOWN  seek -60
#+end_src

***** Shorter Seeking
Do smaller, always exact (non-keyframe-limited), seeks with shift.
Don't show them on the OSD (no-osd).
#+begin_src conf
Shift+RIGHT no-osd seek  1 exact
Shift+LEFT  no-osd seek -1 exact
Shift+UP    no-osd seek  5 exact
Shift+DOWN  no-osd seek -5 exact
#+end_src

***** Subtitle Seeking
#+begin_src conf
Ctrl+LEFT   no-osd sub-seek -1 # seek to the previous subtitle
Ctrl+RIGHT  no-osd sub-seek  1 # seek to the next subtitle
Ctrl+Shift+LEFT sub-step -1    # change subtitle timing such that the previous subtitle is displayed
Ctrl+Shift+RIGHT sub-step 1    # change subtitle timing such that the next subtitle is displayed
#+end_src

***** Longer Seeking
#+begin_src conf
PGUP add chapter 1                     # seek to the next chapter
PGDWN add chapter -1                   # seek to the previous chapter
Shift+PGUP seek 600                    # seek 10 minutes forward
Shift+PGDWN seek -600                  # seek 10 minutes backward
#+end_src

***** Revert Seek
#+begin_src conf
Shift+BS revert-seek                   # undo the previous (or marked) seek
Shift+Ctrl+BS revert-seek mark         # mark the position for revert-seek
#+end_src

**** Zoom and Pan
#+begin_src conf
Alt+left  add video-pan-x  0.1         # move the video right
Alt+right add video-pan-x -0.1         # move the video left
Alt+up    add video-pan-y  0.1         # move the video down
Alt+down  add video-pan-y -0.1         # move the video up
Alt++     add video-zoom   0.1         # zoom in
Alt+-     add video-zoom  -0.1         # zoom out
#+end_src

Reset zoom and pan settings
#+begin_src conf
Alt+BS set video-zoom 0 ; set video-pan-x 0 ; set video-pan-y 0
#+end_src

#+end_src

-b sets sector size (default 512)
-c sets cluster size (default 128)
-d use direct disc access for input file
-J reread latest good sector after every error
-O reopen input file after every read error
-r exit after <n> retry passes (-1=infinity) (default 0)
-R reverse the direction of all passes
-T maximum time since last successful read, in seconds?
-v be verbose (a 2nd -v gives more)
